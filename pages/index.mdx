# Nextjs Study!

넥스트js 스터디 with Nextra

### \_meta.json에 대해

- 폴더로 문서의 계층을 구성할 수 있습니다
- 해당 파일로 라우터의 순서와 보여지는 이름을 분리시킬 수 있습니다
- \_meta.json 파일이 작성되어있지 않아도 작동은하지만
- \_meta.json 파일로 문서구조의 순서를 바로잡을 수 있습니다
- \_meta.json 파일은 "실제 폴더-파일명" : "문서계층의 폴더-파일명" 으로 구성됩니다
- 파일과 폴더명은 넥스트라처럼 사용하였습니다 (띄어쓰기:-)

### 이미지첨부에 대해

- 이미지 이름은 '페이지이름-이미지이름'으로 했습니다. 
- 마크다운에 이미지 첨부하는 형식은 \![이미지 설명](./이미지 경로)
- 에셋폴더에 보관하는것보다 문서랑 가까이두면 좋을것같아서요. 다른 의견있으시면 알려주세용
